**Git**

- 분산 버전관리 프로그램
- repository 로컬 저장소
    git init
    .git
    
- Commit
    - working directory
         작업 중인 실제 디렉토리
         untracked
         ⬇ git add
        
    - staging area
        커밋으로 남기거나 특정버전으로 관리하고 싶은 파일
        staged
        ⬇ git commit
        
    - repository
        커밋 저장
        committed

```
git init
git add 파일명.md
git commit -m "커밋명"
git remote add origin url
git push -u origin main
```


```python
git init
git clone url
# 수정
git add .
git commit -m "커밋 명"
git push origin master
```


**Markdown**

텍스트 기반 markup 언어

- #~###### Heading
    글자 키우기용X
    
- *_ List

- [string](url) Link
- ![string](image_url) Image

- ** xxxx** Bold
- *xxxx* Italic
- ~ ~ xxxx ~ ~ Strike out

- ` ` ` code block ‘ ‘ ‘
- ` Inline code block ‘
- - - - Horizontal line
